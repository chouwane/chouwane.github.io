关于tomcat调用shutdown脚本关闭不了解决方案

1.在catalina.sh文件开头加上CATALINA_PID="tomcat.pid"
（$CATALINA_PID变量是一个pid文件，用来存储tomcat运行的pid）

2.在shutdown.sh文件结尾，
  将 
   exec "$PRGDIR"/"$EXECUTABLE" stop "$@"
  改为
   exec "$PRGDIR"/"$EXECUTABLE" stop -force "$@"
2016-03-15

sitemap & rss in Hexo

SiteMap

执行命令 npm install hexo-generator-sitemap --save
(之前我执行命令少了--save，结果就是不生成SiteMap)

配置文件里添加：
sitemap:
    path: sitemap.xml

重启(hexo s)后，访问/sitemap.xml，会发现已经生成

Rss

执行命令 npm install hexo-generator-feed --save

配置文件里添加：
feed:
  type: atom
  path: atom.xml
  limit: 20
  hub:

重启(hexo s)后，访问/atom.xml，会发现已经生成
